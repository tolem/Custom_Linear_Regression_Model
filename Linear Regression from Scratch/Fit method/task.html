Description

Linear regressions are in charge of modeling the relationships between variables by fitting a straight line. As you have already learned, a simple linear model can be represented as: \(\vec{y} = \beta_0 + \beta_1\vec{x}\).

\(\vec{\beta}\) is a vector of the model's coefficients that contains every \(\beta_j\) (in our case \(j=0;1\)). Finding the values of \(\vec{\beta}\) is known as model fitting. Recall how to find \(\vec{\beta}\):

\[X^t\vec{y} = X^tX\vec{\beta}\\ \vec{\beta} = ( X^tX)^{-1}X^t\vec{y}\]

\(\beta_0\) , which is the first component of the \(\vec{\beta}\), is the intercept coefficient. At this stage, we are assuming our model always has an intercept. This assumption is not always true, we will see further how to set up systems of equations without an intercept. Other components of \(\vec{\beta}\) are the model's coefficients, which at this stage is only \(\beta_1\).

The custom linear regression class can take the following form:

class CustomLinearRegression:

    def __init__(self, *, fit_intercept=True):

        self.fit_intercept = ...
        self.coefficient = ...
        self.intercept = ...

    def fit(self, X, y):
        pass

When the fit_intercept parameter is passed as True to the CustomLinearRegression class, the method solves the system of equations as shown above. If fit_intercept=False, it solves it without an intercept. The coefficient and intercept attributes datatypes are numpy array and float respectively. The * argument in the __init__ method signifies that CustomLinearRegression class can take more arguments than we would be using for this project.
